Excel Template Instructions

How to Use This Template

This page provides instructions for creating a valid Excel file for the Payroll Management System. Please follow these guidelines carefully:

  1. Create an Excel file with multiple sheets, one for each company
  2. Name each sheet according to the company name (e.g., "Golana", "Naps", etc.)
  3. Each sheet must contain the required columns for that specific company
  4. Fill in employee data in each row

Golana Company

Sheet name should be: Golana

Required columns:

Card No NAME Bank Transfer
EMP001 John Doe 25000
EMP002 Jane Smith 30000

Naps Company

Sheet name should be: Naps

Required columns:

INTER ID NAME Total Amount
N001 Robert Johnson 28000
N002 Sarah Williams 32000

Other Companies

Follow the same pattern for other companies, using the appropriate column names:

  • Nayana: 'Employee Code', 'Name of the employee', 'Net Payable'
  • SG: 'card no', 'NAME', 'Bank Transfer'
  • Ink: 'Employee Code', 'Name of the employee', 'Bank Transfer'
  • golden eye: 'Emp Code', 'Names', 'CTC'
  • Genius: 'Card No', 'NAME', 'BANK Transfer'
  • Amigo: 'Emp Code', 'Names', 'Bank Statement'
